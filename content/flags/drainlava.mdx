# Drain Lava

Use the `--drainlava (true|false)` (alias `-dl`) flag to to enabled or disable draining lava based on your default configuration.

By default, lava draining is enabled for all modifications. If you need to disable it, use `--drainlava false`.

If you have disabled it in the configs, you can use `--drainlava true` to perform a lava drain for a single modification.
