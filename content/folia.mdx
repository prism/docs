# Folia Support

<Callout type='info'>Available in Prism 4.4 and above.</Callout>

Folia support is currently in **beta**.

Folia is a multi-threaded Minecraft server from the PaperMC team. Folia assigns regions of chunks to a specific thread, so Prism has to handle rollbacks and restores using those region-specific threads.
