# Basics

Prism offers an API for plugins to interact with, allowing your plugin to:

- Register custom actions.
- Record custom actions as activities.
- Query recorded activities from storage.
- Initiate modifications.

<Callout type='info'>Available in Prism 4.4 and above.</Callout>

## Example

We have published a simple example plugin: [https://github.com/prism/prism/tree/main/prism-paper-example-plugin](https://github.com/prism/prism/tree/main/prism-paper-example-plugin)

## Setup

Configure your java project to reference the prism api dependency.

Be sure to add "prism" to your plugin's `depend` list in `plugin.yml` to ensure the API is available at runtime.

<Tabs items={['gradle', 'maven']}>
  <Tabs.Tab>
```groovy
dependencies {
  compileOnly("org.prism_mc.prism:prism-paper-api:VERSION")
}

repositories {
  maven { url = 'https://nexus.prism-mc.org/repository/maven-releases/' }
  maven { url = 'https://nexus.prism-mc.org/repository/maven-snapshots/' }
}
```
  </Tabs.Tab>
  <Tabs.Tab>
```xml
<repositories>
  <repository>
      <id>prism-releases</id>
      <url>https://nexus.prism-mc.org/repository/maven-releases/</url>
  </repository>
  <repository>
      <id>prism-snapshots</id>
      <url>https://nexus.prism-mc.org/repository/maven-snapshots/</url>
      <snapshots>
          <enabled>true</enabled>
      </snapshots>
  </repository>
</repositories>

<dependencies>
  <dependency>
      <groupId>org.prism_mc.prism</groupId>
      <artifactId>prism-paper-api</artifactId>
      <version>VERSION</version>
      <scope>provided</scope>
  </dependency>
</dependencies>
```
