# /pr storage-config


<Command command="storage-config write-hikari" />

<Callout type='info'>Storage/database changes need a reboot to apply and do not have a `reload` command.</Callout>

Writes a `hikari.properties` file to the `storage` directory using the current configuration.

This is intended to be a starting point for a custom HikariCP configuration.

<Callout type='info'>Hikari-only configurations do not support the "spy" debugging feature.</Callout>
