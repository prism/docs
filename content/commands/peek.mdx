# /pr peek

<Command command="peek (activity id)" />

Items are frequently too complex to represent in chat alone due to lore, enchantments, etc.

Peek will show you an inventory representation of an item given an item action activity id.

Activity IDs can be found by doing a `--nogroup` lookup.

This user interface is visual only and is not a rollback/restore.
