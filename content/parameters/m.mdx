# Materials

Use the `m:` parameter to list one or more item materials.

- `m:[material]`

Supports multiple (comma seperated)!

`m:stone_pickaxe` will query activities that involve stone pickaxes.

See [Material](https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/Material.html).

Also see [itag](../tag).

For blocks, see [b](../b).
