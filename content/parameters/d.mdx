# Duration

<Callout type='info'>This parameter is exclusively used with the proximity command. Available in Prism 4.4 and above.</Callout>

Use the `d:[time code]` parameter to confine the search to activities that occurred within the given duration.

For example, `d:5m` would include records logged within five minutes of the basis activity.

For help with time codes, see [Timecodes](/misc/timecodes).
