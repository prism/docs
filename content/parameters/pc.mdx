# Players Cause

Use the `pc:` parameter to list one or more players that caused an action. This is simply an alias for `p:[playername]`.

- `pc:[playername]`

Supports multiple (comma separated)!

Searches for activities caused by a player.

`pc:viveleroi` will query activities in which `viveleroi` was the cause.
