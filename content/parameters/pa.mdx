# Affected Players

Use the `pa:` parameter to list one or more players affected by an action.

- `pa:[playername]`

Supports multiple (comma separated)!

Searches for activities that affected a player.

`pa:viveleroi` will query activities in which `viveleroi` was affected by an action.
