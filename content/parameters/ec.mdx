# Entity Cause

Use the `ec:[entitytype]` parameter to list one or more entity types that caused an action.

Supports multiple (comma seperated)!

`e:creeper` will query activities that were caused by creepers.

See [Entity Types](https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/entity/EntityType.html)

Also see [etag](../etag).
