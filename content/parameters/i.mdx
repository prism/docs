# Items

Use the `i:` parameter to list one or more item materials.

- `i:[material]`

Supports multiple (comma seperated)!

`i:stone_pickaxe` will query item activities that involve stone pickaxes.

See [Material](https://hub.spigotmc.org/javadocs/bukkit/org/bukkit/Material.html).

Also see [itag](../itag).

For blocks, see [b](../b).
